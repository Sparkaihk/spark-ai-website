# Spark AI Website

Premium enterprise website foundation for **Spark AI Technology Limited**.

Positioning: **AI-Era Data Asset Infrastructure Operator**

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn-style UI primitives
- Framer Motion
- Lucide Icons

## Included

- Project architecture for a scalable Next.js app
- Dark Spark AI design system
- Reusable typography, buttons, cards, containers, sections, and animations
- Premium responsive navbar
- Enterprise footer
- World-class hero section
- Homepage V0.1
- Animated particle network and global data-flow background

## Design Direction

Inspired by Apple, NVIDIA, and Palantir:

- Dark enterprise interface
- High-contrast data visuals
- Precise spacing and typography
- Subtle motion
- Premium infrastructure positioning

## Brand Colors

- Spark Blue: `#007AFF`
